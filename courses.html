100-Level

First Year

The foundational year establishes core programming skills and mathematical maturity. Both modules are compulsory for all BSc Computer Science focal areas.

2 modules · 32 credits
CS 114
Sem 1 16 cr

Introduction to Computer Science

An introduction to computational thinking and programming using Python. Covers control flow, functions, basic data structures, and problem decomposition. Designed for students with no prior programming experience.

Core
CS 144
Sem 2 16 cr

Imperative Programming and Algorithms

C programming, recursion, sorting and searching algorithms, and an introduction to algorithm complexity analysis. Continues the programming thread established in CS 114.

Core

200-Level

Second Year

Second-year core modules deepen the treatment of algorithms and introduce the systems layer of computing. Both modules are required across all BSc focal areas.

2 modules · 32 credits
CS 214
Sem 1 16 cr

Computer Organisation and Systems

Computer architecture, assembly language, memory hierarchies, and the interface between hardware and software. Provides the systems context for higher-level programming.

Core
CS 244
Sem 2 16 cr

Algorithms and Data Structures

Systematic study of fundamental data structures — trees, heaps, graphs, hash tables — and algorithm design techniques including dynamic programming, greedy algorithms, and graph search.

Core

300-Level

Third Year

Third-year modules cover the core systems and theory areas of Computer Science. CS 313 and CS 314 are compulsory across all focal areas; CS 343 and CS 344 are compulsory for General CS and Data Science.

6 modules · 96 credits
CS 313
Sem 1 16 cr

Computer Networks

Network protocols and the TCP/IP stack from the application to the physical layer. Socket programming, routing algorithms, congestion control, and network security fundamentals.

Core
CS 314
Sem 1 16 cr

Concurrent Programming

Multithreaded programming, synchronisation primitives (mutexes, semaphores, monitors), deadlock analysis, and an introduction to parallel programming patterns.

Core
CS 315
Sem 1 16 cr

Systems Programming

Low-level systems programming in C: manual memory management, process control, inter-process communication, and kernel interface programming. Elective for General CS and Data Science focal areas.

Elective
CS 343
Sem 2 16 cr

Operating Systems

OS design and internals: process scheduling, virtual memory, file systems, I/O management, and system calls. Includes case studies from Linux and modern hypervisors.

Core
CS 344
Sem 2 16 cr

Compiler Design

Lexical analysis, parsing, semantic analysis, intermediate code generation, register allocation, and basic optimisation. Students implement a working compiler for a subset of a real language.

Core
CS 345
Sem 2 16 cr

Software Engineering

Software development lifecycle, requirements engineering, design patterns, code quality, testing strategies, and continuous integration. Emphasis on professional practice in team-based settings.

Elective

900-Level · Doctoral

Doctoral Programmes

Doctoral degrees represent the pinnacle of academic achievement. Both the PhD and the DSc require the production of original work that substantially advances knowledge in Computer Science.

2 programmes · 360 credits each
CS 978

Dissertation: Computer Science

The PhD requires a dissertation containing the results of independent research, conducted under close supervision. Candidates work at the absolute frontier of their chosen research area and must make a substantial, original contribution to the field. Requires an MSc in Computer Science or an approved equivalent.

360 credits 3–4 years PhD
CS 998

Published Work: Computer Science

The DSc is the highest academic qualification the department offers. Awarded on the basis of a body of published scientific works of exceptional quality that collectively constitute a substantial enrichment of knowledge in Computer Science. Candidates must already hold a PhD and an established record of scholarly contribution.

360 credits Higher Doctorate DSc
