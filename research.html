  • Formal language theory

Selected Publications

A snapshot of recent research outputs from across the department.

2025 NeurIPS AI & ML

Adaptive Hyper-heuristics for Multi-objective Optimisation in Dynamic Environments

Engelbrecht, A.P., Grobler, T., van Zyl, R.

Introduces a new class of adaptive hyper-heuristics that dynamically select and combine low-level heuristics in response to shifting fitness landscapes, outperforming state-of-the-art methods on standard benchmark suites.

2025 IEEE S&P Security

Scalable Vulnerability Detection via Graph Neural Networks on Code Property Graphs

Yamaguchi, F., Baker Effendi, D., Fischer, B.

Presents a GNN-based approach to vulnerability detection that operates directly on code property graphs, achieving high precision at repository scale without requiring labelled training data.

2025 TACAS Formal Methods

Modular Verification of Concurrent Systems using Compositional Model Checking

Inggs, C., Fischer, B., Visser, W.

A compositional framework for verifying concurrent software that reduces state-space explosion by decomposing global verification tasks into independently solvable sub-problems.

2024 CVPR Computer Vision

COSFIRE-X: Trainable Brain-Inspired Filters for Few-Shot Visual Recognition

Azzopardi, G., Ngxande, M., Hoffmann, M.

Extends the COSFIRE framework with trainable receptive field configurations that enable few-shot generalisation across visual domains with significantly fewer labelled examples.

2024 JMLR AI & ML

On the Sample Complexity of Distribution-Free Learning with Label Noise

Kroon, S.

Establishes tight sample complexity bounds for learning from noisy labels under distribution-free assumptions, with implications for robust model training in low-resource environments.

2024 Semantic Web Journal NLP & Knowledge

Cross-lingual Ontology Alignment for Under-resourced African Languages

Keet, C.M., Pretorius, L.

Proposes a cross-lingual ontology alignment methodology tailored to morphologically rich, low-resource African languages, enabling knowledge reuse across language boundaries in the Semantic Web.
